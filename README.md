## SA-WIKI: Интерактивная база знаний для системных аналитиков

SA-WIKI — это база знаний, созданная специально для системных аналитиков, чтобы быстро повторить теорию перед собеседованиями.

Проект является открытым и каждый может его дополнить. 

Проект и его материалы распространяются по лицензии CC BY 4.0

## Ключевые особенности

• Рендеринг Markdown: Поддержка расширенного синтаксиса Markdown, включая таблицы, списки, а также диаграммы, созданные с помощью Mermaid и PlantUML.

• Подсветка синтаксиса

• Отслеживание прогресса: можно отметить что уже прочитано
