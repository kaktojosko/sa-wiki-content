# Инструменты системного аналитика

Системный аналитик работает на стыке бизнеса, разработки, тестирования и эксплуатации. Поэтому набор инструментов должен покрывать не только сбор требований, но и проверку API, анализ данных, работу с документацией и диагностику проблем.

## Анализ и исследование

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| Кукуха | Думать головой | Думать головой всегда полезно |
| Google | Поиск доменной и технической информации | Базовый вход для исследований и факт-чекинга |
| ChatGPT / LLM | Черновики требований, разбор материалов, генерация вопросов | Использовать с обязательной верификацией |
| Miro | Discovery-сессии и совместный анализ | Удобен для совместной работы |

## Документация и знания

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| Confluence | База знаний и требования | Стандарт в корпоративной среде |
| Notion | Личная/командная база знаний | Удобный для заметок |
| Markdown + Git | Документирование | Основа практик DocOps |

## Моделирование

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| Draw.io | Диаграммы UML/BPMN/архитектура | Универсальный и бесплатный инструмент |
| PlantUML / Mermaid | Создание диаграмм кодом | Когда пишешь диаграмму кодом лучше работает голова |
| BPMN.io / Stormbpmn| Моделирование бизнес-процессов | Это инструмент для BPMN. Тут нечего добавить |

## Управление задачами

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| Jira | Таск-трекер | Инструмент для хранения задач |
| Trello | Лёгкий kanban | Подходит для простых задач |

## API и интеграции

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| Postman | Тестирование API и коллекции запросов | Стандарт де-факто для API-проверок |
| Swagger Editor | Работа с OpenAPI | Проектирование и проверка контрактов |
| Browser DevTools | Анализ сетевых вызовов в браузере | Посмотреть какие запросы уходят из браузера |

## Работа с данными

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| DBeaver | Универсальный SQL-клиент | Поддерживает большинство СУБД |
| Navicat | SQL-клиент и администрирование | Сильный desktop-инструмент для БД |
| dbdiagram.io | Визуализация схем БД | Удобно для создания ERD|

## DevOps и эксплуатация

| Инструмент | Назначение | Комментарий |
| :--- | :--- | :--- |
| Git | Версионирование артефактов аналитики | Ветки, review, трассировка изменений |
| CI/CD для документации | Автопроверка и публикация документации | Практика DocOps |
| Мониторинг (Prometheus, Grafana, ELK) | Анализ инцидентов и метрик | Смотреть логи |
